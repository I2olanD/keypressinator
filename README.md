## This is keypressinator
Keypressinator lets you show your keycodes while your typing. **IN REAL TIME**.

## Hosted here
[keypressinator](https://keypressinator.firebaseapp.com/)

Install dependencies
```bash
yarn
```

Run it
```bash
yarn dev
```
